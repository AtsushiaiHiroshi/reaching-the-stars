# Matriz de publicacion

| Fuente | Uso interno | Distribucion | Condicion |
| --- | --- | --- | --- |
| Lore original de Alcanzando las Estrellas | Si | Si | Declarar autoria y licencia elegida |
| Reglas ORC identificadas por Paizo | Si | Si | Cumplir ORC y aviso de producto |
| Marcas o logo de compatibilidad Starfinder | Si | Condicional | Cumplir la Licencia de Compatibilidad de Paizo |
| Texto de ambientacion y arte de Paizo | Si | No por defecto | Requiere permiso/licencia especifica |
| SRD 5.2.1 de D&D | Si | Si | CC BY 4.0 y atribucion requerida |
| Contenido de D&D fuera del SRD | Si como investigacion | No | Requiere licencia separada |
| 5e.tools | Indice de busqueda | No como fuente copiada | Verificar siempre contra fuente licenciada |
| PI de Riot/League of Legends | Comparacion privada | No | La politica de Riot prohibe usar su PI en juegos o apps sin licencia |
| Otras franquicias | Comparacion privada | No por defecto | Nombres descriptivos en bibliografia; no adaptar expresion protegida |

Esta matriz organiza el trabajo y no sustituye asesoria juridica.
