# Politica de fuentes y licencias

## Jerarquia

1. Starfinder Segunda Edicion oficial.
2. Starfinder Primera Edicion oficial.
3. Pathfinder Segunda Edicion compatible.
4. Material de referencia abierto, como SRD 5.2.1.
5. Diseno original comprobado mediante pruebas.

## Archivo de investigacion

Peliculas, series, novelas, comics y videojuegos pueden registrarse para estudiar tropos. Registrar una obra no concede permiso para adaptar sus nombres, personajes, especies, lugares, arte, texto o combinaciones distintivas.

5e.tools se utiliza solamente como indice para localizar una publicacion. Todo dato distribuible debe comprobarse contra una fuente con licencia adecuada.

## Estados obligatorios

- oficial;
- adaptacion;
- inspirado;
- original;
- pendiente de revision;
- no distribuible.

Cada elemento debe señalar obra, edicion, pagina o URL, titular, licencia, revisor y estado de pruebas.
