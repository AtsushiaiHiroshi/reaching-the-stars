## Resumen

Se creó la plantilla maestra para el Mapa Planetario, una pieza central del Atlas Galáctico.

La plantilla consolida el enfoque macro → micro:

1. Planeta.
2. Mapa global.
3. Regiones.
4. Biomas.
5. Amenazas.
6. Recursos.
7. Puntos de interés.
8. Culturas y civilizaciones derivadas del entorno.

---

## Decisiones registradas

- El mapa planetario se diseña antes de cerrar culturas, política o aventuras.
- Cada mapa puede tener múltiples capas: físico, climático, hidrográfico, biomas, ecológico, amenazas, político, recursos y narrativo.
- El mapa debe explicar cómo el planeta condiciona a razas, especies, variantes, culturas y conflictos.
- Debe servir tanto para documentación del Atlas como para juego en mesa y Foundry VTT.
